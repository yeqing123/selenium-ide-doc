简介

Selenium IDE 是一款 web UI 自动化测试工具，是基于录制和回放操作的火狐浏览器的插件。本教程是官方资料的翻译。

如果不喜欢阅读文字教程，可以尝试学习 Selenium IDE web 自动化测试视频课程，并完成相应的通关任务。
